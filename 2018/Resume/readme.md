
## JSON Resume

_[https://jsonresume.org/](https://jsonresume.org/)_


### Install RESUME

```bash
sudo apt install npm nodejs-legacy
sudo npm install -g resume-cli
```

### Init

```bash
cd ~
mkdir Resume
cd Resume
resume init
```

### Edit

```bash
nano resume.json
```

### Export html

```bash
resume export --theme onepage resume_onepage.html
```

### Make PDF
* Open resume_onepage.html in web browser
* Print to pdf
